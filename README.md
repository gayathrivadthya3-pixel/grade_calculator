# grade_calculator
Student Grade Calculator using Python

## Project Overview
This project is a Python-based Student Grade Calculator that takes student marks as input and assigns a grade (A, B, C, D, F) along with an encouraging message.

## Objectives
- Practice Python basics
- Understand conditional statements (if-elif-else)
- Implement input validation
- Use functions and loops

## Grading Logic
| Marks Range | Grade |
|------------|-------|
| 90-100     | A     |
| 80-89      | B     |
| 70-79      | C     |
| 60-69      | D     |
| 0-59       | F     |

## Technologies Used
- Python 3

## How to Run
1. Install Python
2. Download the project
3. Run the file:

```bash
python grade_calculator.py
```
